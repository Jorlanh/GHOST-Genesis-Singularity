## 1. Briefing da Missão

O **GHOST SYSTEM** é uma interface HUD tática de alto desempenho, projetada para interação de próxima geração. Utiliza uma filosofia de UI "Stealth" (Preto OLED) e integra-se a uma arquitetura distribuída de microsserviços.

**Objetivo Primário:** Fornecer uma interface "God Mode" fluida e controlada por voz para o operador (Senhor Walker).

## 2. Stack Tecnológico (O Arsenal)

Este projeto é construído com tecnologias frontend de nível de elite:

- **Núcleo:** React 18 + Vite (Ferramenta de build ultra-rápida)
- **Linguagem:** TypeScript (Protocolo de tipagem estrita)
- **Estilização:** Tailwind CSS (Framework utility-first)
- **Animação:** Framer Motion (Animações complexas baseadas em física)
- **Componentes UI:** Shadcn-UI + Lucide React (Iconografia tática)
- **Runtime:** Node.js

## 3. Protocolo de Implantação (Local)

Para inicializar o sistema localmente, siga estes procedimentos estritos:

### Pré-requisitos
- Node.js & npm instalados.

### Sequência de Execução

```sh
# Passo 1: Clonar o repositório seguro
git clone <SUA_URL_GIT>

# Passo 2: Entrar no diretório Genesis
cd GHOST-Genesis-singularity

# Passo 3: Instalar dependências táticas
npm install

# Passo 4: Inicializar o Horizonte de Eventos Singular (Servidor Dev)
npm run dev
```
